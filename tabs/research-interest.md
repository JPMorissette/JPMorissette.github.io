<ol>
  <li> The aim of <b>Distributed Analysis</b> is to enable statistical analyses of data partitioned across what are referred to as data nodes. An analytical dataset is said to be horizontally distributed when each data node holds a different subset of individuals for whom the same variables have been recorded. In a similar way, an analytical dataset is said to be vertically distributed when each data node holds a different subset of variables recorded for the same set of individuals. I'm interested in extending existing distributed methods and developing new ones.  </li>
  <li> Keeping in mind that <b>Data Privacy</b> is of utmost importance when both developing and using distributed methods, we must ensure that a given method can meet an organization's privacy requirements. Otherwise, that method will most likely end up on a shelf collecting dust. Finding a reliable, flexible, and easy-to-understand way to deal with privacy is key to enabling collaborative scientific research. </li>
</ol>
